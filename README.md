# Dax Vibes by Geniux

A simple website to play five of Dax's best songs directly in the browser.

## How to Use

1. Obtain MP3 files for the following Dax songs:
   - Lonely Dirt Road
   - Dear God
   - Book of Revelations
   - Joker
   - She Cheated Again

2. Rename the files to:
   - `song1.mp3` (Lonely Dirt Road)
   - `song2.mp3` (Dear God)
   - `song3.mp3` (Book of Revelations)
   - `song4.mp3` (Joker)
   - `song5.mp3` (She Cheated Again)

3. Upload these five MP3 files to the root of this repository (same directory as `index.html`).

4. The site will automatically play the songs via the HTML5 audio player.
